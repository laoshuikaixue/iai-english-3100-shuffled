# IAI English 3100 词乱序版

将 IAI English 发布的《高中英语〈新课程标准〉3100 词总表（2025 版）》重新随机排序，生成适合单词自测、复习和打印使用的乱序 PDF。

本项目只调整词条的排列顺序，不修改原词表中的单词、音标、词性、释义、派生词和星号标记。

## 项目特点

- 按完整词条进行乱序，避免拆散跨行或跨页的释义。
- 保留原文件开头的【编写说明】。
- 保留必修词 `*`、选择性必修词 `**` 等原始标记。
- 使用楷体中文和 Times New Roman 英文、音标排版。
- 在 PDF 末页标注 `Powered By LaoShui`。
- 明确注明单词原始来源为 `IAI English`。
- 使用固定随机种子，默认情况下可重复生成相同的排列结果。
- 生成前后会计算词条内容摘要，确认乱序过程没有修改词条内容。

## 文件说明

| 文件 | 说明 |
| --- | --- |
| `2025届高考英语《新课程标准》3100词总表.pdf` | IAI English 原始词表 PDF |
| `make_shuffled_3100.py` | 词条提取、乱序和 PDF 生成脚本 |
| `2025届高考英语新课程标准3100词总表_乱序版_LaoShui.pdf` | 已生成的乱序版本 |

## 环境要求

- Python 3.10 或更高版本
- [PyMuPDF](https://pymupdf.readthedocs.io/)
- [ReportLab](https://www.reportlab.com/opensource/)

安装 Python 依赖：

```bash
pip install pymupdf reportlab
```

当前脚本默认使用 Windows 自带字体：

- `C:\Windows\Fonts\STKAITI.TTF`
- `C:\Windows\Fonts\STSONG.TTF`
- `C:\Windows\Fonts\times.ttf`
- `C:\Windows\Fonts\timesbd.ttf`

如果在 Linux 或 macOS 上运行，请在 `make_shuffled_3100.py` 中将字体路径替换为本机可用的中文楷体、宋体和 Times 系列字体。

## 生成乱序版

在项目目录中运行：

```bash
python make_shuffled_3100.py \
  "2025届高考英语《新课程标准》3100词总表.pdf" \
  "2025届高考英语新课程标准3100词总表_乱序版_LaoShui.pdf"
```

PowerShell 示例：

```powershell
python .\make_shuffled_3100.py `
  '.\2025届高考英语《新课程标准》3100词总表.pdf' `
  '.\2025届高考英语新课程标准3100词总表_乱序版_LaoShui.pdf'
```

## 自定义乱序结果

脚本默认使用固定随机种子：

```text
LaoShui-IAI-English-3100-2026-07-15
```

传入不同的 `--seed` 即可生成另一套乱序结果：

```bash
python make_shuffled_3100.py input.pdf output.pdf --seed "my-random-seed"
```

相同的原始 PDF 和随机种子会生成相同的词条顺序，方便协作、复现和版本管理。

## 实现方式

脚本会依次完成以下处理：

1. 从原始 PDF 中提取正文的视觉文本行。
2. 识别普通词条、带星号词条、无斜线音标缩写和跨页释义。
3. 将同一词条的全部内容组合为不可拆分的词条块。
4. 使用指定随机种子打乱词条块顺序。
5. 重新排版并生成带封面、页码、署名和来源说明的 PDF。
6. 对乱序前后的词条内容计算 SHA-256 摘要，确保内容集合一致。

## 关于词条数量

“3100 词”沿用原始词表的名称。脚本当前从源文件中识别出 `3093` 个完整词条块，这是因为部分变体词、英美拼写、组合词或关联词共用同一个词条，词条块数量不一定等于标题中的单词数量。

脚本不会删除未被单独识别为新词条的文本；这些内容仍会保留在所属的完整词条块中。

## 编写说明

> 本词汇表共3000词，按字母顺序，依次分类包括：  
> (1) 初中英语义务教育阶段要求掌握的1600个单词；  
> (2) 高中英语必修课程应学习和掌握500个单词（词尾加有一个*号）；  
> (3) 高中英语选择性必修课程应学习和掌握l000个单词（词尾加有两个*号）。  
> 注：本表出现的单词要求全部背诵，并要求掌握其派生词！  
> 如：sight-sighted；space-spacious  
> 新高考命题以新课标为准！

以上内容保留自原始 PDF。乱序版本不再按照字母顺序排列。
